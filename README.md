### 🧬 SONIC — Network Immunisation Algorithm
> Graph neural networks meet epidemic modelling

Designed a network immunisation algorithm that combines **DINO spectral radius minimisation** with **DeepTrace GNN-based source detection**.

- **27% performance gain** over baselines across 3 real-world datasets (1,288–34,671 nodes)
- GraphSAGE + LSTM trained on 750 synthetic epidemic graphs → **96.7% top-5 source detection accuracy**
- Reduced spectral radius by **4.56** using only 3 nodes vs. 100 required by degree centrality
